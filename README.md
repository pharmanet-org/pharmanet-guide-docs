# PharmaNet Help Center

Public documentation for PharmaNet — a platform connecting customers with verified pharmacies.

## Content

- **Customer Guide** — step-by-step instructions for browsing, ordering, and managing your account
- **Seller Guide** — complete guide for pharmacy owners managing their business
- **አማርኛ** — Full Amharic translations of all customer and seller content

## Tech

Built with [Mintlify](https://mintlify.com). Pages are MDX with YAML frontmatter. Configuration in `docs.json`.

## Development

```bash
npm i -g mint
mint dev
```

Preview at `http://localhost:3000`.

## Deployment

Deployed via Vercel. Changes pushed to the default branch are deployed automatically.
