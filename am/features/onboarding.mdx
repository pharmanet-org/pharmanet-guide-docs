# የመጀመሪያ ተሞክሮ (Onboarding)

መተግበሪያው ለመጀመሪያ ጊዜ ሲከፈት 4 የማስተዋወቂያ ገፆች ያሳያል፦

1. **ሰፊ የመድኃኒት ክምችት** — ሁሉንም የሐኪም መድኃኒቶች እና OTC መድኃኒቶች በአንድ ቦታ ያግኙ
2. **ፈጣን እና አስተማማኝ ማድረስ** — መድኃኒቶችዎን ወደ ቤትዎ በእውነተኛ ጊዜ ትከታተል ያግኙ
3. **የሐኪም መድኃኒት በቀላሉ ይስቀሉ** — የሐኪም መድኃኒትዎን ፎቶ ያንሱ ፋርማሲስቶች ያረጋግጣሉ
4. **ጤና በጣትዎ ጫፍ** — የመድኃኒት ማስታወሻዎችን ያዘጋጁ፣ ከፋርማሲስቶች ጋር ያማክሩ

ከተዘለሉ በኋላ እንደ እንግዳ ወደ **መነሻ** ገፅ ይገባሉ።
