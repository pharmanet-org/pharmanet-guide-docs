## ከመጀመርዎ በፊት

**ምን ያስፈልግዎታል፦**
- የሚሰራ ኢሜይል
- የፋርማሲ ቢዝነስ ፍቃድ
- የፋርማሲስት ፍቃድ
- የፋርማኔት መተግበሪያ ተጭኗል

---

## 1. የተጠቃሚ መለያ ይፍጠሩ

1. መተግበሪያውን ይክፈቱ። የ **መገለጫ** ትሩን ይንኩ
2. እንግዳ ከሆኑ **ግባ / መመዝገብ** ይንኩ
3. **ተመዝገብ** የሚለውን ይምረጡ
4. ኢሜይል እና የይለፍ ቃል (ቢያንስ 6 ቁምፊ) ያስገቡ
5. **ተመዝገብ** ይንኩ

## 2. የፋርማሲ ምዝገባ ይጀምሩ

1. ወደ **መገለጫ** ይሂዱ
2. **እንደ ፋርማሲስት ይመዝገቡ** የሚለውን ይንኩ
3. የምዝገባ ቅጹ ይከፈታል

## 3. የምዝገባ ቅጹን ይሙሉ

### የፋርማሲ መገለጫ

| መስክ | መግለጫ |
|-------|----------|
| **የፋርማሲ ስም** | የተመዘገበ የንግድ ስም |
| **የፋርማሲ አድራሻ** | መተየብ ሲጀምሩ አቅጣጫዎች ይታያሉ |
| **የንግድ ስልክ** | ንቁ የሆነ የስልክ ቁጥር |
| **የንግድ ኢሜይል** | የሙያ ኢሜይል |
| **የፋርማሲ ፍቃድ ቁጥር** | ይፋዊ የፍቃድ ቁጥር |
| **መግለጫ** | ስለ ፋርማሲዎ አጭር መግለጫ |

### የባንክ ሂሳብ ዝርዝሮች

| መስክ | መግለጫ |
|-------|----------|
| **የባንክ ስም** | ለምሳሌ፡ የኢትዮጵያ ንግድ ባንክ |
| **የሂሳብ ስም** | የሂሳብ ባለቤት ስም |
| **የሂሳብ ቁጥር** | የባንክ ሂሳብ ቁጥር |

### የሚፈለጉ ሰነዶች

| ሰነድ | አስፈላጊ? |
|-------|------------|
| **የፋርማሲ ፍቃድ** | አዎ — ፎቶ ይስቀሉ |
| **የፋርማሲስት ፍቃድ** | አዎ — ፎቶ ይስቀሉ |
| **ሎጎ** | አይ — አማራጭ |

## 4. ማስገባት

1. **ተመዝገብ** የሚለውን ይንኩ
2. "ተሳክቷል" የሚል መልእክት ይታያል — "ምዝገባ ቀርቧል። አስተዳዳሪው ይገመግማል።"
3. **እሺ** ይንኩ

> ከተላከ በኋላ የፋርማሲ ሂሳብዎ **በመጠበቅ ላይ** ነው። አስተዳዳሪ ሰነዶችዎን ይገመግማል።

## 5. የፋርማሲስት ዳሽቦርድ መድረስ

ወደ **መገለጫ** ይሂዱ → **የፋርማሲስት ዳሽቦርድ** ወይም **የመለያ ሁኔታ ይመልከቱ** ይንኩ።

|--------|--------------|
| **ጸድቋል (Approved)** | የፋርማሲስት ዳሽቦርድ (5 ትሮች) |
| **በመጠበቅ ላይ (Pending)** | "ማረጋገጫ በሂደት ላይ" ገፅ |
| **ተቀባይነት አላገኘም (Rejected)** | ውድቅ ማድረጊያ ምክንያት ያለው ገፅ |
