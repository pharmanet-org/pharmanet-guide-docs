# ትዕዛዞችዎን ይከታተሉ

ይህ ገጽ ትዕዛዞችዎን እንዴት መፈተሽ እና መላኪያቸውን መከታተል እንደሚችሉ ያሳይዎታል።

## ትዕዛዞችዎን ይመልከቱ

1. ከስር ያለውን የ **Orders** ትር ይጫኑ
2. ሁሉም ትዕዛዞችዎ ተዘርዝረው ያያሉ

ትዕዛዞችን ለመደርደር ከላይ ያሉትን **የማጣሪያ ምርጫዎች** ይጠቀሙ፦

| ማጣሪያ | የሚያሳየው |
|---|---|
| **ሁሉም** | ያስቀመጧቸው ትዕዛዞች ሁሉ |
| **መክፈል** | ክፍያ የሚጠብቁ ትዕዛዞች |
| **መላክ** | በፋርማሲ እየተዘጋጁ ያሉ ትዕዛዞች |
| **ተልከዋል** | ወደ እርስዎ በመጓዝ ላይ ያሉ ትዕዛዞች |
| **ደርሰዋል** | የተቀበሏቸው ትዕዛዞች |
| **ተሰርዘዋል** | የተሰረዙ ትዕዛዞች |
| **ተመልሰዋል** | የመለሷቸው ትዕዛዞች |

> **ስክሪን ሾት ያስፈልጋል፦** የትዕዛዞች ገጽ ከማጣሪያ ምርጫዎች ጋር

## የትዕዛዝ ሁኔታዎች

እያንዳንዱ ትዕዛዝ የት እንዳለ የሚነግርዎት የሁኔታ ምልክት አለው፦

| ሁኔታ | ትርጉሙ |
|---|---|
| **በመጠባበቅ ላይ** | ፋርማሲው ትዕዛዝዎን ተቀብሎ እየገመገመ ነው |
| **በማስኬድ ላይ** | ፋርማሲው እቃዎችዎን እያዘጋጀ ነው |
| **ተልኳል** | ትዕዛዝዎ በመላክ ላይ ነው |
| **ደርሷል** | ትዕዛዝዎ ደርሷል |
| **ተሰርዟል** | ትዕዛዙ ተሰርዟል |

## የትዕዛዝ ዝርዝሮችን ይፈትሹ

1. የዝርዝር ገጹን ለመክፈት በማንኛውም ትዕዛዝ ላይ ይንኩ
2. የሚከተሉትን ያያሉ፦
   - **የትዕዛዝ ቁጥር**
   - **የትዕዛዝ ቀን**
   - **የታዘዙ እቃዎች** (ከፎቶዎች እና ብዛቶች ጋር)
   - **ጠቅላላ ዋጋ**
   - **የመላኪያ አድራሻ**
   - **የክትትል ሰንሰለት** — ከትዕዛዝ እስከ መላኪያ ያለውን ጉዞ ያሳያል
   - **የፋርማሲ ስም** — እነሱን ለማግኘት ይንኩ

> **ስክሪን ሾት ያስፈልጋል፦** የትዕዛዝ ዝርዝር ገጽ ከክትትል ሰንሰለት ጋር

## ትዕዛዝ ይሰርዙ

ትዕዛዙ ካልተላከ መሰረዝ ይችላሉ፦

1. የትዕዛዝ ዝርዝር ገጹን ይክፈቱ
2. **ትዕዛዝ ይሰርዙ** ን ይጫኑ
3. መሰረዝ መፈለግዎን ያረጋግጡ
4. የትዕዛዝ ሁኔታው ወደ **ተሰርዟል** ይቀየራል

> **ማስታወሻ፦** ትዕዛዙ ቀድሞ ከተላከ በመስመር ላይ መሰረዝ አይችሉም። እርዳታ ለማግኘት ፋርማሲውን ወይም ድጋፍን ያግኙ።

## መላኪያ ያረጋግጡ

ትዕዛዝዎ ሲደርስ፦

1. ሁሉም እቃዎች ትክክል መሆናቸውን ያረጋግጡ
2. ለማንኛውም ጉዳት ይፈትሹ
3. የትዕዛዝ ዝርዝር ገጹን ይክፈቱ
4. **መላኪያ ያረጋግጡ** ን ይጫኑ

ይህ ፋርማሲው ትዕዛዝዎን እንደተቀበሉ ያሳውቃል።
