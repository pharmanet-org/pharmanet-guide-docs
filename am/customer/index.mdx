# የደንበኛ መመሪያ

እንኳን ወደ ፋርማኔት በደህና መጡ! ይህ መመሪያ እንደ ደንበኛ ማድረግ የሚችሉትን ነገር ሁሉ ያሳይዎታል።

## ምን ማድረግ ይችላሉ

| ርዕስ | የሚማሩት |
|---|---|
| [አካውንት ይፍጠሩ](/am/customer/create-account) | ይመዝገቡ፣ ይግቡ፣ የይለፍ ቃልዎን ያቀናብሩ |
| [ምርቶች ይፈልጉ](/am/customer/find-products) | ይፈልጉ፣ በምድብ ያስሱ፣ ባርኮድ ይቃኙ፣ ማጣሪያዎችን ይጠቀሙ |
| [ፋርማሲዎችን ይጎብኙ](/am/customer/visit-pharmacies) | በአቅራቢያ ያሉ ፋርማሲዎችን ይፈልጉ፣ ደረጃዎችን ይመልከቱ፣ ሰዓቶችን ያረጋግጡ |
| [ትዕዛዝ ያስቀምጡ](/am/customer/place-order) | ወደ ጋሪ ያክሉ፣ ማዘዣ ያስገቡ፣ በቻፓ ይክፈሉ |
| [ትዕዛዞችዎን ይከታተሉ](/am/customer/track-orders) | የትዕዛዝ ሁኔታ ይፈትሹ፣ መላኪያ ይከታተሉ፣ ትዕዛዞችን ይሰርዙ |
| [አካውንትዎን ያስተዳድሩ](/am/customer/manage-account) | መገለጫ ያስተካክሉ፣ ቋንቋ ይቀይሩ፣ የደህንነት ቅንብሮች |

## ለመጀመር

አዲስ ከሆኑ በ[ፈጣን መመሪያ](/am/quickstart) ይጀምሩ።
