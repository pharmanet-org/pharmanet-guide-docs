# የደንበኛ መገለጫ

የታችኛውን የ **መገለጫ** ትሩን ይንኩ።

## መገለጫ ገፅ (ከተገቡ)

- **የመገለጫ ራስጌ** — አምሳያ፣ ሰላምታ ከስምዎ ጋር፣ ስልክ ቁጥር
- **የትእዛዝ ስታቲስቲክስ** — 4 ሳጥኖች፦ **ለመክፈል**፣ **ለመላክ**፣ **ተልኳል**፣ **ደርሷል**
- **ፈጣን ተግባሮች** — 4 አዶዎች፦ **ታሪክ**፣ **የሚወዱ**፣ **የተቀመጡ**፣ **ኩፖኖች**
- **የመጀመሪያ ትዕዛዝ ቅናሽ** — "በመጀመሪያ መድኃኒትዎ 20% ቅናሽ ያግኙ"
- **ታማኝነት ነጥቦች** — "250 ነጥቦች ለመቀየር አለዎት"

## የመገለጫ ክፍሎች

| ክፍል | መግለጫ |
|------|---------|
| **የመገለጫ መረጃ** | ስም፣ የአባት ስም፣ ስልክ ያስተካክሉ — **ለውጦችን አረጋግጥ እና አስቀምጥ** ይንኩ |
| **የተመረጠ ቦታ** | አገር ይምረጡ (ኢትዮጵያ፣ ኬንያ፣ ዲጂቡቲ፣ ሱዳን፣ ሶማሊያ፣ ዩኤስ፣ ካናዳ፣ ዩኬ) |
| **የማድረሻ አድራሻዎች** | አድራሻ ያክሉ፣ ያስተካክሉ፣ ይሰርዙ — **ነባሪ አድራሻ** ማብሪያ |
| **የክፍያ ዘዴዎች** | የተቀመጡ ካርዶችን ይመልከቱ ወይም **አዲስ የክፍያ ካርድ ጨምር** |
| **ቋንቋ** | **English** ወይም **አማርኛ** ይምረጡ — ወዲያውኑ ይለወጣል |
| **ፋርማሲዎን ይመዝገቡ** | እንደ ፋርማሲስት ለመመዝገብ ይንኩ |
| **የመለያ ሁኔታ ይመልከቱ** | የፋርማሲ ምዝገባ ሁኔታ ይፈትሹ |

## ደህንነት እና ግላዊነት

> ⚠️ አብዛኛዎቹ ማብሪያዎች ገና አይሰሩም (የUI ቅድመ እይታ)

**የመለያ ደህንነት፦**
- **የይለፍ ቃል ለውጥ** — በቅርቡ
- **ባዮሜትሪክ መግባት** — FaceID/የጣት አሻራ — በቅርቡ
- **የመግባት እንቅስቃሴ** — በቅርቡ

**ማረጋገጫ፦**
- **ባለሁለት ደረጃ ማረጋገጫ (2FA)** — በቅርቡ
- **ኤስኤምኤስ ማረጋገጫ** — በቅርቡ
- **የኢሜይል ማረጋገጫ** — በቅርቡ

**ግላዊነት፦**
- **የትዕዛዝ ታሪክ ደብቅ** — በቅርቡ
- **የውሂብ ማጋራት** — በቅርቡ
- **የታገዱ ሰዎች** — በቅርቡ

**መሳሪያ፦**
- **ካሽ አጽዳ** — ይሰራል ✅
- **የፍለጋ ታሪክ አጽዳ** — ይሰራል ✅
- **የመተግበሪያ ዝመናዎች** — ይሰራል ✅

## የእርዳታ ማእከል

- **የተለመዱ ጥያቄዎች (FAQ)** — ትዕዛዝ እንዴት ማዘዝ፣ የፋርማኔት ነጥቦች፣ ግላዊነት፣ ትዕዛዝ መሰረዝ
- **ድጋፍ ቻት** — የድጋፍ ቡድን 24/7 ዝግጁ ነው

## ስለ ፋርማኔት

ተልእኳችን፣ ዋና እሴቶች (አስተማማኝነት፣ ፈጣንነት፣ ተደራሽነት) እና የግላዊነት ፖሊሲ ይመልከቱ።

## መውጣት

1. ቀይ **ውጣ** አዝራር ይንኩ
2. "እርግጠኛ ነዎት መውጣት ይፈልጋሉ?" — **ውጣ** ይንኩ ወይም **ይቅር**
