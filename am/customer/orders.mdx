# ትዕዛዞች

## ትዕዛዞችን ማየት

ሁለት መንገዶች አሉ፦

**ዘዴ 1 — ክትትል ትር፦**
1. የታችኛውን **ትእዛዞችን ተከታተል** ትሩን ይንኩ
2. ካልገቡ "ግባ / መለያ ፍጠር" ይታያል
3. ከገቡ የትዕዛዞች ዝርዝር ይታያል

**ዘዴ 2 — ፈጣን ተግባር፦**
በ**መነሻ** ገፅ ላይ **ትዕዛዞች** ይንኩ

## ማጣሪያ

| ማጣሪያ | የሚያሳየው |
|---------|------------|
| **ሁሉም** (All) | ሁሉንም ትዕዛዞች |
| **ለመክፈል** (To Pay) | ክፍያ የሚጠብቁ |
| **ለመላክ** (To Ship) | በሂደት ላይ ያሉ |
| **ተልኳል** (Shipped) | በመጓዣ ላይ ያሉ |
| **ደርሷል** (Delivered) | የተጠናቀቁ |
| **ተሰርዟል** (Cancelled) | የተሰረዙ |

## የትእዛዝ ካርዶች

እያንዳንዱ ትዕዛዝ ያሳያል፦

- **የትእዛዝ ቁጥር** (ለምሳሌ #12345)
- **ጠቅላላ ዋጋ** በብር
- **ሁኔታ ባጅ** — ባለ ቀለም መለያ
- **የማዘዣ ሁኔታ** (አስፈላጊ ከሆነ)
- **ዝርዝሮችን ተመልከት** አዝራር

## የሁኔታ ፍሰት

```
ተቀመጠ → በመጓዣ ላይ → ደርሷል → ተላከ
```

1. **ተቀመጠ** (Placed) — ትዕዛዝ አስቀምጠዋል። ፋርማሲው ገና አላረጋገጠም
2. **በመጓዣ ላይ** (In Transit) — ፋርማሲው እያዘጋጀ ነው
3. **ደርሷል** (Arrived) — እቃዎቹ በመንገድ ላይ ናቸው
4. **ተላከ** (Delivered) — ትዕዛዙ ደርሷል

## የትእዛዝ ዝርዝሮች

ትዕዛዝ ይንኩ ዝርዝሮችን ለማየት፦

- የትዕዛዝ ማጠቃለያ (ቁጥር፣ ቀን፣ ድምር፣ የክፍያ ዘዴ፣ የክፍያ ሁኔታ)
- የደንበኛ መረጃ
- የማድረሻ አድራሻ
- የሐኪም መድኃኒት (ከተሰቀለ)
- የትዕዛዝ እቃዎች (ምስል፣ ስም፣ ብዛት፣ ዋጋ)
- የክትትል ሁኔታ ጊዜ መስመር
