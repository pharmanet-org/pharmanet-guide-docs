# መጀመር

ፋርማኔት መድሃኒቶችን ለማግኘት፣ ለማዘዝ እና ከተረጋገጡ ፋርማሲዎች ለማድረስ የሚያስችል መድረክ ነው።

## ለመጀመሪያ ጊዜ ሲከፈት

1. መተግበሪያውን ይክፈቱ
2. 4 የማስተዋወቂያ ገፆች ይታያሉ፦
   - **ሰፊ የመድኃኒት ክምችት** — ሁሉንም መድኃኒቶች በአንድ ቦታ ያግኙ
   - **ፈጣን እና አስተማማኝ ማድረስ** — እስከ ቤትዎ ይደርሳል
   - **የሐኪም መድኃኒት በቀላሉ ይስቀሉ** — ፎቶ ያንሱ ፋርማሲስቶች ያረጋግጣሉ
   - **ጤና በጣትዎ ጫፍ** — ማሳሰቢያዎችን ያዘጋጁ
3. ከተዘለሉ በኋላ ወደ **መነሻ** ገፅ ይደርሳሉ

## ያለ መግቢያ ማድረግ የሚችሏቸው

- **መነሻ** ገፅን ማሰስ (ባነሮች፣ ምድቦች፣ ታዋቂ ምርቶች፣ ቅርብ ፋርማሲዎች)
- **ፋርማሲ** ትርን ማሰስ (ሁሉም ፋርማሲዎች፣ መገለጫዎች)
- የምርት ዝርዝሮችን ማየት (ምስሎች፣ ዋጋ፣ መግለጫ)
- **ፍለጋ** መጠቀም
- ወደ **ጋሪ** ማከል

## መግባት የሚያስፈልጋቸው

- **ጋሪ** ትር → "መግባት ያስፈልጋል"
- **ክትትል** ትር → "ግባ / መለያ ፍጠር" (signInCreateAccount)
- **መገለጫ** ትር → ሁሉም እቃዎች ተቆልፈዋል
- **ክፍያ** (Checkout / Pay Now) → መግባት ያስፈልጋል
- **የሚወዱ** (ዊሽሊስት) → መግባት ያስፈልጋል
- **ውይይት** እና **ግምገማ** → መግባት ያስፈልጋል

## መለያ መፍጠር

1. መተግበሪያውን ይክፈቱ
2. **መገለጫ** ትሩን ይንኩ → **ግባ / መመዝገብ** ይንኩ
3. **ተመዝገብ** ይምረጡ
4. ኢሜይል እና የይለፍ ቃል ያስገቡ
5. **ተመዝገብ** ይንኩ

## በመተግበሪያው ውስጥ መሰረታዊ አሰሳ

የታችኛው የአሰሳ አሞሌ 5 ትሮች አሉት፦

| አዶ | ስያሜ (ከመተግበሪያው) | ገፅ |
|-----|----------------------|------|
| መነሻ | **መነሻ** | ዋና ገፅ |
| ሆስፒታል | **ፋርማሲ** | ሁሉም ፋርማሲዎች |
| ጋሪ | **ጋሪ** | የጋሪ ይዘቶች |
| ሰዓት | **ትእዛዞችን ተከታተል** | የትእዛዝ ሁኔታ |
| ተጠቃሚ | **መገለጫ** | የሂሳብ ቅንብሮች |
