# ፈጣን ጅምር

## ደንበኛ (በፍጥነት ለመጀመር)

1. መተግበሪያውን **ፋርማኔት** ይክፈቱ
2. 4 የማስተዋወቂያ ገፆችን ይዘልሉ (ወይም ይመልከቱ)
3. ወደ **መነሻ** ገፅ ይደርሳሉ — ምንም መግቢያ ሳያስፈልግ ማሰስ ይችላሉ
4. ለማዘዝ፦ **ተመዝገብ** ይንኩ → ኢሜይል እና የይለፍ ቃል ያስገቡ → **ተመዝገብ** ይንኩ
5. መድሃኒቶችን መፈለግ እና ማዘዝ ይጀምሩ

## ፋርማሲስት (ለመመዝገብ)

1. የተጠቃሚ መለያ ይፍጠሩ (ከላይ እንደተጠቀሰው)
2. ወደ **መገለጫ** ይሂዱ → **እንደ ፋርማሲስት ይመዝገቡ** ይንኩ
3. የፋርማሲ መረጃ ያስገቡ፦ ስም፣ አድራሻ፣ ፍቃድ ቁጥር፣ የባንክ ዝርዝሮች
4. የፋርማሲ እና የፋርማሲስት ፍቃድ ፎቶ ይስቀሉ
5. **ተመዝገብ** ይንኩ ለማስገባት
6. አስተዳዳሪ እስኪያረጋግጥ ድረስ ይጠብቁ
