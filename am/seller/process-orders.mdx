# ትዕዛዞችን ያስኬዱ

ይህ ገጽ የደንበኛ ትዕዛዞችን እንዴት ማስተናገድ እንደሚችሉ ያሳይዎታል።

## ትዕዛዞችዎን ይመልከቱ

**በሞባይል ላይ፦**
1. የፋርማኔት አፕሊኬሽኑን ይክፈቱ እና በፋርማሲ አካውንትዎ ይግቡ
2. የ **Orders** ትር ይጫኑ
3. ትዕዛዞችን ለመደርደር ከላይ ያሉትን **የማጣሪያ ምርጫዎች** ይጠቀሙ

**በድር ዳሽቦርድ ላይ፦**
1. ወደ ሻጭ ዳሽቦርድ ይግቡ
2. በግራ ምናሌ ውስጥ **ትዕዛዞች** ን ይጫኑ
3. ሁሉንም ትዕዛዞች በተመሳሳይ የማጣሪያ አማራጮች ይመልከቱ

## የትዕዛዝ ሁኔታ ፍሰት

ደንበኛ ትዕዛዝ ሲያስቀምጥ በነዚህ ደረጃዎች ውስጥ ያልፋል፦

```
በመጠባበቅ ላይ → ተረጋግጧል → በማስኬድ ላይ → ተልኳል → ደርሷል
```

## ደረጃ 1፦ አዲስ ትዕዛዝ ይገምግሙ

1. አዲስ ትዕዛዝ ሲደርስ በ**በመጠባበቅ ላይ** ይታያል
2. ትዕዛዙን ይክፈቱ እና ያያሉ፦
   - **የደንበኛ ስም** እና **መገኛ**
   - **የታዘዙ እቃዎች** ከብዛቶች ጋር
   - **ጠቅላላ ዋጋ**
   - **የመላኪያ አድራሻ**
   - **ማዘዣ** (አስፈላጊ ከሆነ) — የተሰቀለውን ምስል ይመልከቱ
   - **የትዕዛዝ ማስታወሻዎች** ከደንበኛው

## ደረጃ 2፦ ትዕዛዙን ያረጋግጡ

1. ትዕዛዙን ይገምግሙ እና ማዘዣውን ይፈትሹ (አስፈላጊ ከሆነ)
2. ሁሉም እቃዎች በክምታት ላይ መሆናቸውን ያረጋግጡ
3. **ትዕዛዝ ያረጋግጡ** ወይም **Accept** ን ይጫኑ
4. ትዕዛዙ ወደ **በማስኬድ ላይ** ይሄዳል

ትዕዛዙን መፈጸም ካልቻሉ፦
1. **ትዕዛዝ ይሰርዙ** ን ይጫኑ
2. ምክንያት ይምረጡ
3. ደንበኛው ይነገራል እና ገንዘቡ ይመለሳል

## ደረጃ 3፦ ትዕዛዙን ያዘጋጁ

1. ሁሉንም እቃዎች ይሰብስቡ
2. በጥንቃቄ ያሽጉ
3. አስፈላጊ ሰነዶችን ያክሉ (ደረሰኝ፣ መመሪያዎች)
4. እየዘጋጁ እያለ ትዕዛዙ በ **በማስኬድ ላይ** ላይ ይቆያል

## ደረጃ 4፦ እንደተላከ ምልክት ያድርጉ

1. ትዕዛዙ ዝግጁ ከሆነ እና ለማጓጓዣ አገልግሎት ከተሰጠ፦
2. **እንደተላከ ምልክት ያድርጉ** ን ይጫኑ
3. የ**አጓጓዥ ስም** ያስገቡ (የመላኪያ አገልግሎት)
4. የ**ክትትል ቁጥር** ያስገቡ (ካለ)
5. ትዕዛዙ ወደ **ተልኳል** ይሄዳል
6. ደንበኛው ማሳወቂያ ይደርሰዋል

## ደረጃ 5፦ መላኪያ

1. ደንበኛው ትዕዛዙን ይቀበላል
2. ደንበኛው መላኪያውን ሲያረጋግጥ ትዕዛዙ ወደ **ደርሷል** ይሄዳል
3. ማረጋገጫ ካለዎት እርስዎም **ደርሷል** ብለው ምልክት ማድረግ ይችላሉ

## ትዕዛዝ መሰረዝ

ከመድረሱ በፊት በማንኛውም ጊዜ ትዕዛዝ መሰረዝ ይችላሉ፦

1. ትዕዛዙን ይክፈቱ
2. **ትዕዛዝ ይሰርዙ** ን ይጫኑ
3. ምክንያት ይምረጡ፦
   - ክምታት አልቋል
   - ደንበኛው ጠይቋል
   - ማዘዣ ልክ ያልሆነ
   - ሌላ
4. ደንበኛው ይነገራል
5. ክፍያው ለደንበኛው ይመለሳል
