# ምርቶችን ያስተዳድሩ

ይህ ገጽ ምርቶችዎን በፋርማኔት ላይ እንዴት ማስተዳደር እንደሚችሉ ያሳይዎታል።

## ምርቶችዎን ይመልከቱ

**በሞባይል ላይ፦**
1. የፋርማኔት አፕሊኬሽኑን ይክፈቱ እና በፋርማሲ አካውንትዎ ይግቡ
2. የ**ምርቶች** ትር ወይም ከዳሽቦርድዎ **ክምታት** ን ይጫኑ
3. የሁሉም ምርቶችዎ ዝርዝር ያያሉ
4. የተወሰነ ምርት ለማግኘት **የፍለጋ አሞሌ**ን ይጠቀሙ
5. በምድብ ወይም በክምታት ሁኔታ ለማጣራት **ማጣሪያዎች**ን ይጠቀሙ

**በድር ዳሽቦርድ ላይ፦**
1. ወደ ሻጭ ዳሽቦርድ ይግቡ
2. በግራ ምናሌ ውስጥ **ምርቶች** ን ይጫኑ
3. የተሟላ የምርት ዝርዝር ከፍለጋ እና ማጣሪያ አማራጮች ጋር ያያሉ

## አዲስ ምርት ያክሉ

1. **ምርት ያክሉ** ን ይጫኑ
2. የምርት ዝርዝሮችን ይሙሉ፦

| መስክ | ምን ማስገባት |
|---|---|
| **የምርት ስም** | የመድሃኒት ወይም የምርት ስም |
| **መግለጫ** | አጭር መግለጫ (ምን እንደሆነ፣ ምን እንደሚሰራ) |
| **ምድብ** | ከዝርዝሩ ይምረጡ (የህመም ማስታገሻ፣ አንቲባዮቲኮች፣ ወዘተ) |
| **ዋጋ** | ደንበኛው የሚከፍለው መጠን |
| **የክምታት ብዛት** | ምን ያህል አሃዶች እንዳሉዎት |
| **መጠን** | ጥንካሬው ወይም መጠኑ (ለምሳሌ፦ 500ሚግ) |
| **አምራች** | ምርቱን የሚሰራው ኩባንያ |
| **የማለቂያ ቀን** | ምርቱ የሚያልቅበት ቀን |
| **ማዘዣ ያስፈልጋል** | ደንበኞች ማዘዣ ከፈለጉ ያብሩ |
| **ፎቶዎች** | እስከ 5 የምርት ፎቶዎችን ያስገቡ |

3. **Save** ወይም **ምርት ያክሉ** ን ይጫኑ

## ምርት ያስተካክሉ

1. ምርቱን በዝርዝርዎ ውስጥ ያግኙት
2. **የእርሳስ አዶ**ን (አስተካክል) ይጫኑ
3. ማንኛውንም ዝርዝር ይቀይሩ
4. ለውጦችዎን ለማስቀመጥ **Save** ን ይጫኑ

## ምርት ይሰርዙ

1. ምርቱን በዝርዝርዎ ውስጥ ያግኙት
2. **የቆሻሻ መጣያ አዶ**ን (ሰርዝ) ይጫኑ
3. መሰረዝዎን ያረጋግጡ
4. ምርቱ ከካታሎግዎ ይወገዳል እና ደንበኞች ከእንግዲህ አያዩትም

## የክምታት ሁኔታ ቀለሞች

እያንዳንዱ ምርት የክምታት ሁኔታውን ያሳያል፦

- **አረንጓዴ** — በክምታት ላይ (በቂ ብዛት)
- **ቢጫ** — ዝቅተኛ ክምታት (እያለቀ ነው፣ በቅርቡ ያዘዙ)
- **ቀይ** — አልቋል (ምንም አይገኝም)

## ዝቅተኛ ክምታት ማንቂያዎች

የምርት ክምታት ሲያልቅ፦
1. በዳሽቦርድዎ ላይ **ማሳወቂያ** ያያሉ
2. ምርቱ **ቢጫ** የክምታት አመልካች ያሳያል
3. ከመጠናቀቁ በፊት ከአቅራቢዎችዎ ተጨማሪ ክምታት ያዘዙ
