## Open Your Profile

1. In the **Pharmacist Dashboard**, tap the **Profile** tab (last icon — user icon)
2. The profile page has these sections:
   - **Header** — Pharmacy logo, name, pharmacist name, approval status badge
   - **Order Management** — Quick stats with counts
   - **Quick Actions** — Shortcuts to key features
   - **Menu sections** — Settings and info pages

## Profile Header

**If signed in:**
- Top-left **"Pharmacist Portal"** badge
- Pharmacy logo (or hospital icon placeholder)
- Pharmacy name
- Pharmacist name
- Approval status badge: **Approved** (green), **Pending** (orange), or **Rejected** (red)
- Top-right icons: Notification bell (→ Notifications page), Settings gear (→ Security & Privacy)

**If guest:**
- "Join as Pharmacist" heading
- Description text
- **Sign In / Register** button

## Menu: Account & Pharmacy

### Personal Information
1. Tap **Personal Information**
2. Edit your **First Name**, **Last Name**, or **Phone Number**
3. Tap the profile image to change it — choose **Gallery** or **Camera**
4. Tap **Confirm & Save Changes**
5. Green toast: *"Profile updated successfully"*

### Pharmacy Profile
1. Tap **Pharmacy Profile** (highlighted in blue)
2. Update any of:
   - **Store Name** — Pharmacy business name
   - **Store URL Slug** — URL-friendly identifier
   - **Business Phone** — Contact number
   - **Description** — About your pharmacy
   - **Business Address** — Type to search with autocomplete. Tap a suggestion to set coordinates
   - **Set Coordinates** — Tap to open the map picker. Tap the map to drop a pin at your location
3. Tap **Save Changes**
4. Green toast: *"Profile updated successfully"*

## Switch to Customer View

From the pharmacist Profile page, tap **Switch to Customer View** to go back to the regular customer dashboard. This lets you browse as a customer, view products, and place test orders.

## Log Out

1. Scroll to the bottom of the Profile page
2. Tap the red **Log Out** button
3. Confirmation dialog: *"Are you sure you want to logout?"*
4. Tap **Log Out** to confirm

## Troubleshooting

| Problem | What to do |
|---------|------------|
| Can't save pharmacy profile | Check all required fields (Store Name, Store URL Slug) |
| Map picker not working | Ensure location permissions are granted in your device settings |
| Address autocomplete not showing | Type at least 3 characters |
| Approval status shows Pending | Wait for admin review. Check back later |
| Can't change license number | License info is collected during registration only. Contact support to update |
