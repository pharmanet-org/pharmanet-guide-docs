## Open Your Inventory

1. Log into the **Pharmacist Dashboard**
2. Tap the **Inventory** tab (second icon — box icon)
3. The page shows your medication catalog with a search bar at the top

## Search Medications

- Tap the search bar (hint: *"Search by name..."*)
- Type a medication name — results filter as you type
- Pull down to refresh the list

## Medication Cards

Each medication card shows:

| Element | Description |
|---------|-------------|
| **Image** | 80x80 product image, or a placeholder icon |
| **Name** | Product name |
| **Dosage • Category** | e.g., "500mg • Antibiotics" |
| **Rx badge** | Red "Rx" label if prescription is required |
| **Price** | In ETB |
| **Stock quantity** | Number of units. Turns red if at or below low-stock threshold |
| **Trash icon** | Tap to delete the medication |

Tap any card to edit it. Tap the trash icon to delete.

## Add a New Medication

1. Tap the **+** FAB button (blue, bottom-right corner)
2. Fill in the form:

### Basic Information

| Field | Required | Notes |
|-------|----------|-------|
| **Medication Name** | Yes | |
| **Description** | No | Optional details |
| **Category** | Yes | Select from a dropdown of available categories |
| **Price** | Yes | Numeric value in ETB |
| **Stock Quantity** | Yes | Numeric value |

### Medical Specifications

| Field | Notes |
|-------|-------|
| **Dosage** | e.g., 500mg, 10ml, 1 tablet |
| **Manufacturer** | Brand or manufacturer name |
| **Expiry Date** | Tap the field to open a date picker. Future dates only |
| **Prescription Required** | Toggle ON if customers must upload a prescription |

### Images

- Tap the **+** box to select multiple images from your gallery
- Existing images (if editing) show as thumbnails
- Images are scrolled horizontally

3. Tap **Add Medication** to save

## Edit a Medication

1. Tap any medication card in the list
2. The same form opens, pre-filled with existing data
3. Update any fields
4. Tap **Save Changes**

## Delete a Medication

1. Tap the **trash icon** on a medication card
2. A confirmation dialog appears: *"Delete Medication"*
3. Tap **Delete** (red) to confirm, or **Cancel**
4. Green toast: *"Medication deleted"*

## Empty State

If no medications exist, you'll see:
- A search box icon
- Text: *"No medications found"*
- **Add Medicine** button to add your first product

## Low Stock Alerts

On the **Home** tab, the **Low Stock** stat shows the count of medications at or below their threshold (default: 5 units). Tap **See All** to jump to Inventory.

The dashboard also shows a **Low Stock Alert** section listing your most critical items.

## Troubleshooting

| Problem | What to do |
|---------|------------|
| Can't find the add button | Look for the blue **+** button at bottom-right |
| Product not saving | Check all required fields (Name, Price, Stock) |
| Stock shows wrong number | Edit the product and update the quantity, then save |
| Error loading categories | Check your internet connection. Pull down to retry |
