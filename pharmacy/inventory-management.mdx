## ⚠️ Troubleshooting

| Problem | What to do |
| :--- | :--- |
| Can't find "Add Product" button | Make sure you're logged in as Pharmacist (not Customer) |
| Product not saving | Check all required fields (Name, Price, Stock) |
| Stock shows wrong number | Edit product and update quantity, then save |
| Expired products still visible | Delete or disable them manually |
