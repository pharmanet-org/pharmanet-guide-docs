## Where to Find Orders

In the **Pharmacist Dashboard**, tap the **Track** tab (third icon — receipt icon).

The order list shows with status filter chips at the top.

## Filter Orders

Tap a filter chip to narrow the list:

| Chip | What it shows |
|------|---------------|
| **All** | Every order |
| **Pending** | Orders waiting for you to process |
| **Processing** | Orders you've accepted and are preparing |
| **Dispensed** | Orders packed and ready for shipping |
| **Shipped** | Orders in transit |
| **Delivered** | Completed orders |
| **Cancelled** | Cancelled orders |

## Order Cards

Each order card displays:

- **Order number** (e.g., `#12345`)
- **Date** (formatted: "Jan 15, 2026 14:30")
- **Total amount** (ETB)
- **Status badge** — color-cased label at top-right
- **Details** button — view full order info
- **Action button** — advance to the next status

## Order Status Flow

Orders move through these statuses. The action button label changes at each stage:

| Current Status | Badge Color | Action Button | Next Status |
|----------------|-------------|---------------|-------------|
| **Pending** | Orange | **Process** (blue) | Processing |
| **Processing** | Blue | **Dispensed** (primary) | Dispensed |
| **Dispensed** | Primary | **Shipped** (purple) | Shipped |
| **Shipped** | Purple | **Delivered** (green) | Delivered |
| **Delivered** | Green | None (terminal) | — |
| **Cancelled** | Red | None (terminal) | — |

## Process an Order

1. Find an order with **Pending** status
2. Tap the **Process** button on the card
3. A confirmation dialog appears: *"Mark order #[number] as Processing?"*
4. Tap **Yes** to confirm
5. The status updates immediately
6. Repeat at each stage (Process → Dispensed → Shipped → Delivered)

## View Order Details

1. Tap **Details** on any order card
2. The order detail page shows:

### Order Info
- Order number and date
- Payment method (e.g., Chapa, T-chapa)
- Payment status (e.g., PAID, PENDING)
- Total amount (ETB)

### Customer Info
- Full name and email
- Shipping address

### Prescription
Only shown if the order contains prescription-required items:
- Uploaded prescription image
- Prescription status badge (Pending / Approved / Rejected)
- Tap status to change it

### Order Items
- Each product with image, name, quantity (`Qty: 2`), and price

### Management Section
- **Order Status** — Dropdown to manually change status (pending/processing/dispensed/shipped/delivered/cancelled)
- **Prescription Status** — Dropdown (only if prescription exists): pending/approved/rejected
- **Carrier** — Text field for delivery company name
- **Tracking Number** — Text field for tracking code
- **Admin Notes** — Multi-line text field for internal notes

### Save Changes
1. Make your changes in the management section
2. Tap the **Update Order** button at the bottom
3. Green toast: *"Order updated successfully"*

## Cancel an Order

1. Open the order detail page
2. In the **Order Status** dropdown, select **Cancelled**
3. Tap **Update Order**
4. Confirm the change

## Order Stats Grid

On the **Profile** tab, the **Order Management** section shows real-time stats:

| Stat | Description |
|------|-------------|
| **New** | Recently placed orders |
| **Pending** | Orders awaiting action |
| **Packed** | Orders dispensed and ready |
| **Shipped** | Orders in transit |
| **Cancelled** | Cancelled orders |

Tap any stat to jump to a filtered order list.

## Dashboard Order Sections

On the **Home** tab:

- **Recent Prescription Orders** — Shows the 3 most recent orders with quick status badges
- **Low Stock Alert** — Shows medications running low on stock

## Communicate with Customers

1. From the dashboard, tap the **Chats** tab (fourth icon — message icon)
2. Active conversations appear with customer name, last message, time, and unread count
3. Tap a chat to open the conversation
4. Send text messages or attach images (camera/gallery)
5. Product inquiry chats show a banner with the product name and photo

## Troubleshooting

| Problem | What to do |
|---------|------------|
| Order not showing up | Pull down to refresh the list |
| Can't see the action button | Terminal statuses (Delivered, Cancelled) don't have action buttons |
| Wrong status displayed | Pull down to refresh |
| Error updating status | Check your internet connection and try again |
