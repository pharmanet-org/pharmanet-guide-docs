# Customer Guide

Welcome to PharmaNet! This guide will walk you through everything you can do as a customer.

## What You Can Do

| Topic | What You'll Learn |
|---|---|
| [Create Your Account](/customer/create-account) | Sign up, log in, reset your password |
| [Find Products](/customer/find-products) | Search, browse categories, scan barcodes, use filters |
| [Visit Pharmacies](/customer/visit-pharmacies) | Find nearby pharmacies, see ratings, check hours |
| [Place an Order](/customer/place-order) | Add to cart, upload prescriptions, pay with Chapa |
| [Track Your Orders](/customer/track-orders) | Check order status, track delivery, cancel orders |
| [Manage Your Account](/customer/manage-account) | Edit profile, change language, security settings |

## Getting Started

If you are new to PharmaNet, start with the [Quick Start Guide](/quickstart).

> **Need help right now?** Go to the **Help & Support** section or contact us through the app.
