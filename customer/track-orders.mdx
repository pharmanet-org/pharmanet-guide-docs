# Track Your Orders

This page shows you how to check your orders and track their delivery.

## View Your Orders

1. Tap the **Orders** tab at the bottom of the screen
2. You will see all your orders listed

Use the **filter chips** at the top to sort orders:

| Filter | Shows |
|---|---|
| **All** | Every order you have placed |
| **To Pay** | Orders waiting for payment |
| **To Ship** | Orders being prepared by the pharmacy |
| **Shipped** | Orders on their way to you |
| **Delivered** | Orders you have received |
| **Cancelled** | Orders that were cancelled |
| **Returns** | Orders you have returned |

> **Screenshot needed:** Orders page with filter chips

## Order Statuses

Each order has a status badge that tells you where it is:

| Status | What It Means |
|---|---|
| **Pending** | The pharmacy has received your order and is reviewing it |
| **Processing** | The pharmacy is preparing your items |
| **Shipped** | Your order is on its way for delivery |
| **Delivered** | Your order has arrived |
| **Cancelled** | The order was cancelled |

## Check Order Details

1. Tap any order to open its detail page
2. You will see:
   - **Order number**
   - **Order date**
   - **Items ordered** (with photos and quantities)
   - **Total price**
   - **Delivery address**
   - **Tracking timeline** — shows the journey from order to delivery
   - **Pharmacy name** — tap to contact them

> **Screenshot needed:** Order detail page with tracking timeline

## Track Your Delivery

If your order is shipped:

1. On the order detail page, scroll to the **Tracking** section
2. See the **timeline** with the current status highlighted
3. If available, see the **delivery person's information**
4. You may also see the **delivery location on a map**

## Cancel an Order

You can cancel an order if it has not been shipped yet:

1. Open the order detail page
2. Tap **Cancel Order**
3. Confirm that you want to cancel
4. The order status will change to **Cancelled**

> **Note:** If the order has already been shipped, you cannot cancel it online. Contact the pharmacy or support for help.

## Confirm Delivery

When your order arrives:

1. Check that all items are correct
2. Check for any damage
3. Open the order detail page
4. Tap **Confirm Delivery**

This lets the pharmacy know you received your order.

## Returns

If you need to return an item:

1. Go to the order detail page
2. Tap **Return**
3. Select the items you want to return
4. Give a reason for the return
5. Follow the instructions provided
