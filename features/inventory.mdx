## Real-time Inventory Visibility

PharmaNet replaces fragmented supply chains with a unified, real-time shared product catalog. Dealers and Sellers publish their stock levels, making them instantly visible across the verified network.

### Catalog Features

- **Live Inventory Counts**: Stock levels update instantly upon sale or manual adjustment.
- **Expiry Risk Mitigation**: Track batch numbers and expiry dates to avoid high-risk medicine sales.
- **Shared Visibility**: Dealers can publish catalogs directly to pharmacies, eliminating manual order delays.

### Inventory Lifecycle

1. **Publish Stock**: Sellers/Dealers add medicines with batch IDs and quantity.
2. **Catalog Distribution**: Stock becomes searchable by verified network members.
3. **Low-Stock Triggers**: System alerts notify sellers when essential medicines reach threshold levels.
4. **Verified Sales**: Transactions through the network ensure inventory counts stay synchronized.

## Advanced Search Visibility

Members can search the catalog using advanced filters like **Category (e.g., Antibiotics, Vitamins)**, **Manufacturer**, and **Proximity**.
